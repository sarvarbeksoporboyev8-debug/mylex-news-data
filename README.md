# Lex.uz News Data

Auto-updated news data from lex.uz for the Lex.uz AI mobile app.

## Files

| File | Language |
|------|----------|
| `news_uz_Cyrl.json` | Uzbek (Cyrillic) |
| `news_uz.json` | Uzbek (Latin) |
| `news_ru.json` | Russian |
| `news_en.json` | English |
| `metadata.json` | Update info |

## Updates

This repo is automatically updated daily at 6:00 AM UTC by GitHub Actions.

New documents are added on top, historical data is preserved.
